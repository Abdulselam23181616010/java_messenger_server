Bu bir messenger uygulamanın sunucu kısmıdır, gönderileri alıp çözer ve cevap verir.
Gereksinimler:
  Socket
  Swing
  MySql

Request Code:
  1 - Login request
  2 - sign up request
  3 - send message request
  4 - load previous messages request

Response Code:
  [requestCode]0 - olumsuz
  [requestCode]1 - olumlu
